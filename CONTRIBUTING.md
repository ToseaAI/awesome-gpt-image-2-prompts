# Contributing

Thanks for helping grow the collection! This repo is auto-generated from a
single source of truth, so the workflow is intentionally narrow.

## The one rule

**Only edit `prompts.json`.** The `README.md` and `README.zh-CN.md` files are
generated by `scripts/build_readmes.py` and will be overwritten on merge.

## Adding a new prompt

1. Drop the output image into `images/NN.jpg` where `NN` is the next
   zero-padded index (e.g. `61.jpg`).
2. Append a new entry to `prompts.json` under `entries`:

   ```json
   {
     "index": 61,
     "id": "61-short-kebab-slug",
     "title": { "en": "Short title", "zh": "简短标题" },
     "category": "profile",
     "author": {
       "displayName": "Name on tweet",
       "xHandle": "handle_without_at"
     },
     "sourceUrl": "https://x.com/handle/status/...",
     "image": "images/61.jpg",
     "description": { "en": "One sentence.", "zh": "一句话。" },
     "prompt": "Full prompt text, verbatim.",
     "promptType": "prose",
     "featured": false,
     "publishedAt": "2026-04-22",
     "toseaUrl": "https://tosea.ai/prompts/gpt-image-2#61-short-kebab-slug"
   }
   ```

3. Run the generator locally:

   ```bash
   python scripts/build_readmes.py
   ```

4. Commit the JSON edit, the image, and the regenerated READMEs in the
   same PR. (The CI workflow will double-check that READMEs are in sync.)

## Field notes

- **`category`** — one of: `profile`, `poster`, `infographic`, `social`,
  `comic`, `ui`, `thumbnail`. Pick the closest fit; we merge rather than
  split.
- **`author.xHandle`** — omit if you cannot verify. We do not fabricate
  handles; missing handles render as plain-text author names.
- **`sourceUrl`** — prefer the author's own tweet/post. Short links
  (`t.co/...`) are acceptable.
- **`featured`** — reserved for editors. Set `false` in PRs unless a
  maintainer asks otherwise.
- **`prompt`** — verbatim. Do not paraphrase, translate, or reformat.

## Translations

We ship English + Simplified Chinese. If the original prompt is in a
different language, put the original in `prompt` and provide translated
`title` / `description` in `en` / `zh`.

## Code of conduct

Be kind, credit authors, and do not submit prompts you did not create or
verifiably discover from a public post.
